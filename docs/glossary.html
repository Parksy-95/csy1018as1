Glossary

Please see below Key Words used in industry:

Key Words Word Defination
HTML Stands for "Hypertext Markup Language." HTML is the language used to create webpages.
CSS CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen
Class The class attribute specifies one or more class names for an element
ID The id attribute specifies a unique id for an HTML element
HREF The HREF is an attribute of the anchor tag, which is also used to identify sections within a document
Basic Formatting HTML uses elements like < b > and < i > for formatting output
Body The < body > tag defines the document's body.
Children The children property returns a collection of an element's child elements
Parent An HTML element within another HTML element may inherit the design details of the parent or surrounding HTML element
Comment The comment tag is used to insert comments in the source code.
Div The < div > tag defines a division or a section in an HTMLdocument.
Head Definition and Usage. The < head > element is a container for all the head elements.
Headings HTML defines six levels of headings. A heading element implies all the font changes,
paragraph breaks before and after, and any white space necessary to render the heading
Horizontal Rules In HTML 4.01, the < hr > tag represents a horizontal rule. However, the < hr > tag may still be displayed as ahorizontal rule in visual browsers.
Hyperlink Hyperlink is an icon, graphic, or text in a document that links to another file or object
Images The background-image property sets one or more background images for an element.
By default, a background-image is placed at the top-left corner of an element, and repeated both vertically and horizontally.
Linebreaks The line-break CSS property is used to specify how (or if) to break lines when working with punctuation and symbols.
Links The :link CSS pseudo-class represents an element that has not yet been visited. It matches every
unvisited < a > , < area > , or < link > element that has an href attribute
List: Ordered ists Unordered- unordered lists (< ul >), and in menu lists (< menu >).
List: Ordered 18. Ordered list The < li > tag defines a list item. The < li > tag is used inordered lists
Paragraphs The < p > tag defines a paragraph.
Semantic Formatting Semantics is the study of the meanings of words and phrases in a language.
Tables 21. Tables An HTML table is defined with the < table > tag. Each table row is defined with the < tr > tag. A table header is defined with the < th > tag.
Tags A command inserted in a document that specifies how the document
Element An HTML element is an individual component of an HTML document or web page
Title The < title > tag is required in all HTML documents and it defines the title of the document. The < title > element: defines a title in the browser toolbar.
Properties A property, in some object-oriented programming languages, is a special sort of class member, intermediate in functionality between a field
Margin A margin is the space outside something
Font-Family The size and the font of text that will be displayed
Selectors The selector points to the HTML element you want to style.
Attribute Selectors The CSS attribute selector matches elements based on the presence or value of a given attribute.
Child Selectors A child selector matches when an element is the child of some element
HTML5 HTML5- latest version of HTML.
Ruby Ruby is a dynamic, reflective, object-oriented, general-purpose programming language.
Python Python is an interpreted high-level programming language for general-purpose programming
SQL SQL is a domain-specific language used in programming and designed for managing data held in a relational database management system
PHP PHP is a server-side scripting language designed for web development but also used as a general-purpose programming language.
C C is a general-purpose, imperative computer programming language, supporting structured programming, lexical variable scope and recursion,
C++ C++ is a general-purpose programming language. It has imperative, object-oriented and generic programming features, while also providing facilities for
low-level memory manipulation
Java Java is a general-purpose computer programming language that is concurrent, class-based, object-oriented, and specifically designed to have as few
implementation dependencies as possible
W3C Members work together to develop standards set on web pages.
Adobe Photoshop Adobe Photoshop is a raster graphics editor developed and published by Adobe Systems for macOS and Windows
Action Script Designed especially for website animation.
SEO SEO methodology on how much traffic you get to a webpage. The higher the amount of traffic the higher it will appear on a search engine.
Graphic Design Graphic design is the process of visual communication and problem-solving using one or more of typography, photography and illustration.
jQuery The id attribute specifies a unique id for an HTML element Query is a cross-platform JavaScript library designed to simplify the client-side scripting of HTML.
Bash/Shell Bash/Shell is a Unix shell and command language written by Brian Fox for the GNU Project as a free software replacement for the Bourne shell.
Strings In computer science a string is any finite sequence of characters (i.e., letters, numerals, symbols and punctuation marks)
